# Sphere Demo

A compact, design-forward fintech dashboard demo focused on send-money flows, account management, and route comparison. Built as a showcase UI with deterministic mock data.

## Stack

- Next.js (App Router), React, TypeScript
- Tailwind CSS + Shadcn UI
- Motion for minor animations

## Run

```bash
npm install
npm run dev
```

## Notes

- All data is mocked and deterministic.
- No external APIs or persistence.
