# Sphere Demo

A compact, design-forward fintech dashboard demo focused on send-money flows, account management, and route comparison. Built as a showcase UI with deterministic mock data and zero backend dependencies.

## Stack

- Next.js (App Router), React, TypeScript
- Tailwind CSS + Radix UI
- Motion for micro-interactions

## Run

```bash
npm install
npm run dev
```

## Notes

- All data is mocked and deterministic.
- No external APIs or persistence.

## UI Inspiration

- Consumer + Enterprise Fintech: https://wise.com, https://wise.design/
- Dashboard block reference: https://bundui.io/
